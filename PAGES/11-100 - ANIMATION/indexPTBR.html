CSS TIME-FUNCTIONS

ease ( padrão )

Uma animação com o começo lento, então rapido,
então final mais lento

linear

Uma animação com a mesma velocida do inicio para o fim

ease-in

Uma animação com o começo lento

ease-out

Uma animação com o final lento

ease-in-out

Uma animação com um começo e fim lentos

step( 7, start )

Crie um movimento escalonado com sete intervalos distintos,
fazendo a transição a cada movimento, começando do início
do intervalo
