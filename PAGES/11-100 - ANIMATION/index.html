CSS TIME-FUNCTIONS

ease(This is default)

An animation with a slow start, then fast, then end slowly

linear

An animation with the same speed from start to end

ease-in

An animation with a slow start

ease-out

An animation with a slow end

ease-in-out

An animation with a slow start and end

step( 7, start )

Create a stepped motion with seven distinct intervals, transitioning
abruptly at each step, starting at the beginning of the interval
